---

## 🎯 How to Use?
1. Open the Injector.
2. Select your target App/Game.
3. Choose the `.so` (shared library) file you want to inject.
4. Click on **"Install"** to install inside virtual environment.
5. Click on **"Inject"** to launch the application instantly.
